# Prueba tecnica 

 * Analisis Exploratorio.ipynb : muestra el analisis estadistico y de visualizacion realizado sobre el conjunto de datos y a partir del cual se definen las variables a utilizar en el proceso de modelado. 
 
 * Modelado.ipynb : se prueban diferentes modelos matematicos que permiten predecir el nivel de riesgo de no pago con su valides sustentada en el anilisis exploratorio. 
 
 * Product_recognition: punto adicional donde se implementa un modelo de clasificacion de imagenes utilizando rutinas predefinidas con torch y torchvision.
