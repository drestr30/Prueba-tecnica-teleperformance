# Product Recognition 

  * Clasificador de imagenes.ipynb: se define el proceso para el entrenamiento y validacion de un modelo de clasificacion de imagenes especializado en los productos Doria, Jet y Tosh. 
   
  * models.py: arquitecturas de modelos profundos re-implementadas a partir de torchvision
  * read_data.py: rutinas pre-definadas para la lectura y visualizacion de datos. 
  * train.py: rutinas pre-definidas diseñadas para el entrenamientos de modelos profundos en imagenes.
  
  * test.py: rutinas pre-definidas diseñadas para la validacion de modelos profundos en imagenes
